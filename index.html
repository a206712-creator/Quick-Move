About Me

Hello, my name is CHEN HAOJIE. I am a student taking TTTM2213 Mobile Application Development. This portfolio documents my learning journey throughout the semester, covering Android development using Kotlin, Jetpack Compose, MVVM architecture, Room Database, Firebase Firestore, REST APIs, and mobile sensors.


My final application, Quick Move, aligns with SDG 11 by providing ride booking, ride sharing, weather information, and location services to improve transportation accessibility.

Laboratory Activities

Lab 1 – User Interface Design

Developed the initial user interface using Jetpack Compose. Focused on layout arrangement and composable structure.

Reflection:
This lab introduced me to Jetpack Compose and modern Android UI development. I learned how composables work together to create responsive user interfaces. The experience built a strong foundation for future mobile development tasks.

Lab 2 – Interaction and State

Implemented user interactions using TextField, Button and state management.

Reflection:
I learned how state controls UI updates and user interactions. Understanding state management helped me create dynamic applications. This lab improved my understanding of reactive programming concepts.

Lab 3 – Material Design

Applied Material Design principles, themes, typography and cards.

Reflection:
This lab taught me how visual consistency improves user experience. I explored color schemes and typography to create professional-looking interfaces. Material Design greatly enhanced the overall appearance of my app.

Lab 4 – Navigation and ViewModel

Implemented navigation flow and ViewModel architecture.

Reflection:
I learned how ViewModel preserves data across configuration changes. Navigation Compose enabled smooth transitions between screens. This lab introduced me to MVVM architecture.

Lab 5 – Room Database

Integrated Room Database for persistent local storage.

Reflection:
This lab helped me understand local data persistence. I implemented Entity, DAO, Repository and Room Database. The application can now retain data even after restarting.
