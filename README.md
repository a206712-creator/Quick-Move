# 📱 Application Overview

Quick Move is a smart transportation mobile application developed using Kotlin and Jetpack Compose.

The application combines:

* Ride Booking
* Wallet Selection
* Booking Summary
* Booking History
* Community Ride Sharing
* Weather Information
* GPS Location Tracking

into one integrated mobile platform.

---

# ✨ Main Features

| Feature              | Description                            |
| -------------------- | -------------------------------------- |
| 💳 Wallet Selection  | Select transportation wallet package   |
| 🚖 Ride Booking      | Book rides with pickup and destination |
| 📄 Booking Summary   | Review booking before confirmation     |
| 📚 Booking History   | View previous bookings                 |
| 🤝 Community Sharing | Share rides with other users           |
| ☁ Firebase Cloud     | Store ride-sharing information online  |
| 🌦 Weather API       | Display live weather information       |
| 📍 GPS Location      | Display current coordinates            |

Used for:

* Booking History
* Offline Storage
* Persistent Data

Components:

* Entity
* DAO
* Room Database
* Repository
* ViewModel

Benefits:

✔ Data survives app restart

✔ Offline accessibility

✔ Structured storage
